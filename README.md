# Tetris GameHub

Classic Tetris adaptation with neon/retro aesthetics and scoring system for the GameHub platform.

## ✨ Features
- **Retro Aesthetics**: 'Press Start 2P' font and neon colors.
- **Arcade Sounds**: Generated via Web Audio API (no external files).
- **Responsive**: Touch controls for mobile.
- **Dynamic Difficulty**: Speed increases exponentially with completed lines.
- **Highscores**: Integrated with GameHub's `game_api.php`.

## 🕹️ Controls
- **Left / Right Arrow**: Move piece.
- **Up Arrow**: Rotate piece.
- **Down Arrow**: Soft drop.
- **Space**: Hard drop.

## 🚀 Installation
Simply clone the repository and open `index.html` in your browser.

```bash
git clone https://github.com/DustinGitBot/tetris-gamehub.git
```
