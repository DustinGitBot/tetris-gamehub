# Tetris GameHub

Versión adaptada del Tetris clásico con estética neon/retro y sistema de puntuación para la plataforma GameHub.

## ✨ Características
- **Estética Retro**: Fuente 'Press Start 2P' y colores neón.
- **Sonidos Arcade**: Generados mediante Web Audio API (sin archivos externos).
- **Responsive**: Controles táctiles para móvil.
- **Dificultad Dinámica**: La velocidad aumenta exponencialmente con las líneas completadas.
- **Highscores**: Integrado con `game_api.php` de GameHub.

## 🕹️ Controles
- **Flecha Izquierda / Derecha**: Mover pieza.
- **Flecha Arriba**: Rotar pieza.
- **Flecha Abajo**: Caída suave.
- **Espacio**: Caída instantánea (Hard Drop).

## 🚀 Instalación
Simplemente clona el repositorio y abre `index.html` en tu navegador.

```bash
git clone https://github.com/DustinGitBot/tetris-gamehub.git
```
