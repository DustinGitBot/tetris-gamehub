# Tetris GameHub 🕹️🧱

A modern, neon-styled Tetris game built with HTML5 Canvas and Tailwind CSS. Featuring retro sound effects and a dynamic difficulty system.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![HTML5](https://img.shields.io/badge/HTML5-Canvas-orange)](https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API)

## ✨ Features

- 🎮 **Classic Gameplay** - All the standard Tetris mechanics you know and love.
- 🎨 **Neon Aesthetics** - Glowy retro design with 'Press Start 2P' font.
- 🔊 **Web Audio Sounds** - Procedureal sound effects for moves, clears, and game states.
- 📱 **Mobile Friendly** - On-screen touch controls for playing on the go.
- 📈 **Dynamic Difficulty** - Speed increases exponentially as you clear lines.
- 🏆 **High Score System** - Integrated with GameHub backend for top scores.

## 🛠️ Controls

### Keyboard
- **Arrow Left/Right**: Move piece
- **Arrow Up**: Rotate piece
- **Arrow Down**: Soft drop
- **Space**: Hard drop
- **Restart Button**: Start a new game

### Mobile
- **Buttons**: Move, Rotate, and Drop pieces via touch.

## 📦 Installation

Just clone the repository and open `index.html` in your browser. No build steps required!

```bash
git clone https://github.com/DustinGitBot/tetris-gamehub.git
cd tetris-gamehub
open index.html
```
